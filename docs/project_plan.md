# MinecraftSync

## 프로젝트 소개

Minecraft 싱글플레이 월드를 Google Drive와 자동으로 동기화하는 프로그램

## 프로젝트 목적

노트북과 데스크톱에서 하나의 월드를 번갈아 플레이할 수 있도록
Google Drive를 이용하여 월드를 자동으로 동기화한다.

## 개발 언어

- Python 3

## 사용 기술

- Google Drive API
- zipfile
- pathlib
- logging
- json
- PyInstaller

## 개발 목표

- 마인크래프트 종료 후 실행
- 최신 월드 자동 판단
- 자동 업로드
- 자동 다운로드
- 임시 ZIP 자동 삭제
- exe 배포
